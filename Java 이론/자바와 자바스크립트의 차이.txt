1. 
- 자바스크립트는 HTML, 상호작용성과 동적 시각적 효과를 나타내기 위해 필요한 객체 기반의 스크립트 언어이다.
  자바는 클래스 기반의 객체 지향 언어 이다.
  
  
2. 
- 자바는 변수 자료형이 반드시 선언되어야하며 컴파일 과정을 거친다.(소스 코드를 자바 바이트 코드로 변환)
  자바스크립트는 변수 자료형을 미리 선언하지 않아도 브라우저가 자동으로 파악한다.
  또한, 인터프리터 언어로서 컴파일 과정을 거치지 않는다.
  
  
3.
- 자바는 서버/클라이언트 개발 모두 가능하지만
  자바스크립트는 node.js를 제외하면 클라이언트용 언어이다.
